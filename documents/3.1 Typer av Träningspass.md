3.1 Typer av Träningspass
Beskrivning: Detta dokument beskriver de olika typer av träningspass som används i respektive träningsfas inom triathlonträning. Träningspassen är organiserade efter disciplin och kategori, med korta beskrivningar för att underlätta planeringen av adepternas träningsprogram.

1. Översikt över Träningstyper
•	Styrketräning (Weights)
•	AE: Aerob Uthållighet (Aerobic Endurance)
•	TE: Tempo
•	MF: Muskulär Styrka (Muscular Force)
•	SS: Snabbhetsfärdigheter (Speed Skills)
•	ME: Muskulär Uthållighet (Muscular Endurance)
•	AC: Aerob Kapacitet (Aerobic Capacity)
•	SP: Sprintkraft (Sprint Power)
•	BW: Brickträning (Brick Workouts)
•	T: Tester (Testing)
<a name="2"></a>
2. Styrketräning
Faser av Styrketräning:
1.	AA – Anatomical Adaptation
o	Syfte: Anpassa muskler och leder för kommande träning.
o	Används i: Förberedelsefasen.
2.	MT – Max Transition
o	Syfte: Övergång till maximal styrketräning.
o	Används i: Slutet av Grundträningsfasen.
3.	MS – Max Strength
o	Syfte: Utveckla maximal styrka.
o	Används i: Byggfasen.
4.	SM – Strength Maintenance
o	Syfte: Bibehålla styrka under tävlingssäsongen.
o	Används i: Toppnings- och Tävlingsfasen.
<a name="3"></a>
3. Simning
<a name="3.1"></a>
3.1 Aerob Uthållighet (AE)
•	AE1: Recovery
o	Lätta simpass för återhämtning och teknikfokus.
•	AE2: Aerobic Endurance Intervals
o	Längre intervaller för att förbättra aerob uthållighet.
<a name="3.2"></a>
3.2 Tempo (TE)
•	TE1: Tempo Intervals
o	Intervaller i högre fart för att öka tempotålighet.
<a name="3.3"></a>
3.3 Muskulär Styrka (MF)
•	MF1: Muscular Force Reps
o	Övningar med fokus på kraft och styrka i armtag.
•	MF2: Open Water Current Intervals
o	Simning i strömmande vatten för ökad styrka.
•	MF3: Paddles
o	Användning av paddlar för att öka motstånd och styrka.
<a name="3.4"></a>
3.4 Snabbhetsfärdigheter (SS)
•	SS1: Fast Forms 25s
o	Korta sprinter med fokus på teknik och snabbhet.
•	SS2: Toys Sets
o	Användning av hjälpmedel som fenor för att förbättra snabbhet.
<a name="3.5"></a>
3.5 Muskulär Uthållighet (ME)
•	ME1: Long Cruise Intervals
o	Längre intervaller i jämnt tempo för uthållighet.
•	ME2: Short Cruise Intervals
o	Korta, intensiva intervaller för att öka muskulär uthållighet.
•	ME3: Threshold
o	Träning vid tröskelfart för att förbättra prestationsförmågan.
<a name="3.6"></a>
3.6 Aerob Kapacitet (AC)
•	AC1: VO2Max Intervals
o	Högintensiva intervaller för att öka syreupptagningsförmågan.
•	AC2: Aerobic Capacity Intervals
o	Intervaller som utmanar den aeroba kapaciteten.
<a name="3.7"></a>
3.7 Tester (T)
•	CSS Test
o	Test för att fastställa Critical Swim Speed.

4. Cykel
<a name="4.1"></a>
4.1 Aerob Uthållighet (AE)
•	AE1: Recovery
o	Lätta cykelpass för återhämtning.
•	AE2: Aerobic Endurance
o	Långa pass i lugnt tempo för att bygga uthållighet.
<a name="4.2"></a>
4.2 Tempo (TE)
•	TE1: Tempo Endurance
o	Längre pass i högre tempo för tempotålighet.
<a name="4.3"></a>
4.3 Muskulär Styrka (MF)
•	MF1: Force Reps
o	Korta, intensiva backintervaller för styrka.
•	MF2: Hilly Ride
o	Cykling i kuperad terräng för ökad styrka.
•	MF3: Hill Repeats
o	Upprepade backintervaller för att bygga muskelstyrka.
<a name="4.4"></a>
4.4 Snabbhetsfärdigheter (SS)
•	SS1: Spin-ups
o	Kadensövningar för att förbättra pedalteknik.
•	SS2: Isolated Leg
o	Träning med ett ben i taget för bättre teknik.
<a name="4.5"></a>
4.5 Muskulär Uthållighet (ME)
•	ME1: Cruise Intervals
o	Intervaller i tröskelfart för uthållighet.
•	ME2: Hill Cruise Intervals
o	Intervaller i backe för att kombinera styrka och uthållighet.
•	ME3: Crisscross Intervals
o	Växlande intensitet för att utmana kroppen.
•	ME4: Threshold Ride
o	Längre pass vid eller nära tröskelintensitet.
<a name="4.6"></a>
4.6 Aerob Kapacitet (AC)
•	AC1: VO2Max Intervals
o	Högintensiva intervaller för att öka syreupptagningsförmågan.
•	AC2: Pyramid Intervals
o	Stegvis ökande och minskande intervaller.
•	AC3: Hill Intervals
o	Intensiva backintervaller för aerob kapacitet.
<a name="4.7"></a>
4.7 Tester (T)
•	FTP-test
o	Test för att fastställa Functional Threshold Power.

<a name="6"></a>
6. Brickträning
Brickträning innebär kombination av två discipliner i ett träningspass, oftast cykel följt av löpning.
<a name="6.1"></a>
6.1 Aerob Uthållighet (BAE)
•	BAE1: Aerobic Endurance Brick
o	Lång cykling följt av löpning i lugnt tempo för uthållighet.
<a name="6.2"></a>
6.2 Tempo (BTE)
•	BTE1: Tempo Brick
o	Cykel och löpning i högre tempo för tempotålighet.
<a name="6.3"></a>
6.3 Snabbhetsfärdigheter (BSS)
•	BSS1: Transition 1 (T1) Practice
o	Öva snabb växling från simning till cykel.
•	BSS2: Transition 2 (T2) Practice
o	Öva snabb växling från cykel till löpning.
<a name="6.4"></a>
6.4 Muskulär Uthållighet (BME)
•	BME1: Muscular Endurance Brick
o	Kombination av cykel och löpning för att öka muskulär uthållighet.
•	BME2: Hilly Brick
o	Cykling och löpning i kuperad terräng för styrka och uthållighet.
<a name="6.5"></a>
6.5 Aerob Kapacitet (BAC)
•	BAC1: Bike-Intervals Brick
o	Intensiva cykelintervaller följt av löpning.
•	BAC2: Run Intervals Brick
o	Cykling följt av intensiva löpintervaller.

<a name="7"></a>
7. Definierade Termer och Förkortningar
•	AE (Aerobic Endurance): Träning för att förbättra kroppens förmåga att använda syre över längre tid.
•	TE (Tempo): Träning i högre fart för att förbättra tempotålighet.
•	MF (Muscular Force): Träning för att öka muskelstyrka.
•	SS (Speed Skills): Övningar för att förbättra teknik och snabbhet.
•	ME (Muscular Endurance): Träning för att förbättra muskulär uthållighet.
•	AC (Aerobic Capacity): Högintensiv träning för att öka syreupptagningsförmågan.
•	SP (Sprint Power): Träning för att förbättra explosivitet och sprintförmåga.
•	BW (Brick Workouts): Kombinationspass av två discipliner, oftast cykel och löpning.
•	T (Testing): Tester för att utvärdera prestation och sätta träningszoner.
