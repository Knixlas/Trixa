3.3 Testvärden
Beskrivning:
Detta dokument används för att samla in viktiga testvärden från adepten. Dessa värden är nödvändiga för att skapa ett så precist och anpassat träningsprogram som möjligt. Värdena kommer att användas för att beräkna träningszoner, fastställa styrketräningsnivåer och optimera träningsbelastningen.

<a name="1"></a>
1. Allmänna Fysiska Värden
•	Längd: Anges i centimeter.
•	Vikt: Anges i kilogram.
•	Blodtryck: Anges i mmHg (t.ex. 120/80).
<a name="2"></a>
2. Cykel- och Löpningstester
•	FTP (Functional Threshold Power): Vad har du för FTP? Ange i watt.
•	Anaerob Tröskelfart (AT eller LT2): Vad har du för anaerob tröskelfart? Ange i km/h eller min/km.
•	Aerob Tröskelfart (LT eller LT1): Vad har du för aerob tröskelfart? Ange i km/h eller min/km.
•	AT-puls: Vad har du för puls vid anaerob tröskel? Ange i slag per minut (bpm).
•	LT-puls: Vad har du för puls vid aerob tröskel? Ange i slag per minut (bpm).
•	CSS (Critical Swim Speed): Vad har du för CSS? Ange i sekunder per 100 meter.
<a name="3"></a>
3. Styrketester
•	Bänkpress: Maxvikt eller antal repetitioner vid en given vikt.
•	Knäböj (Squat): Maxvikt eller antal repetitioner vid en given vikt.
•	Marklyft (Deadlift): Maxvikt eller antal repetitioner vid en given vikt.
•	Axelpress (Overhead Press): Maxvikt eller antal repetitioner vid en given vikt.
•	Pull-ups eller Chins: Antal repetitioner om maxvikt ej är tillgänglig.
•	Roddövning (t.ex. Barbell Row): Maxvikt eller antal repetitioner vid en given vikt.
<a name="4"></a>
4. Instruktioner för Sammanställning
•	Samla in samtliga testvärden från adepten genom att ställa ovanstående frågor.
•	Dokumentera noggrant varje värde för att använda dem i träningsprogrammet.
•	Använd värdena för att justera träningsbelastningen och skapa individuellt anpassade träningszoner och styrketräningsmål.
<a name="5"></a>
5. Sammanfattning
Genom att samla in dessa testvärden från adepten kan du skapa ett mer precist och effektivt träningsprogram. Det hjälper dig att förstå adeptens fysiska förmågor och begränsningar, vilket möjliggör en optimal anpassning av träningsbelastningen.
<a name="6"></a>
6. Definierade Termer och Förkortningar
•	FTP (Functional Threshold Power): Den högsta effekten en cyklist kan upprätthålla i en timme.
•	Anaerob Tröskelfart (AT eller LT2): Den högsta hastigheten som kan upprätthållas innan kroppen övergår till anaerob energiproduktion.
•	Aerob Tröskelfart (LT eller LT1): Hastigheten vid vilken kroppen börjar producera mer mjölksyra än den kan eliminera.
•	CSS (Critical Swim Speed): Den snabbaste hastigheten som en simmare kan upprätthålla utan att bli utmattad.
